# 电商用户行为分析及可视化系统

**管理分析端**

**版本信息**

| 使用库包       | 版本号     | 新特性                       |
| -------------- | ---------- | ---------------------------- |
| vue            | 3.0.0      | compositionAPI响应式数据     |
| vue-router     | 4.0.1      | 动态路由控制解决用户登录权限 |
| vuex           | 4.0.0-rc.2 | 数据缓存                     |
| ant design vue | 2.0.0-rc.3 | admin-UI                     |
| echarts        | 5.0.2      | 提供admin图表                |

**安装**

```bash
yarn install
```

**运行**

```bash
yarn serve
```

卸载node_modules

```
rimraf node_modules
```
